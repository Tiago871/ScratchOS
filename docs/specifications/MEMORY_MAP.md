# ScratchOS Memory Map

## Real Mode

| Address | Description |
|----------|-------------|
| 0x00000 | Interrupt Vector Table |
| 0x00400 | BIOS Data Area |
| 0x07C00 | ScratchBoot Stage1 |
| 0x07E00 | ScratchBoot Stage2 |
| 0x90000 | Free Memory |
